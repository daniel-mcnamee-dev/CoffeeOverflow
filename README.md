# CoffeeOverflow

![Node.js](https://img.shields.io/badge/Node.js-Backend-green)
![Express](https://img.shields.io/badge/Express.js-Framework-lightgrey)
![Handlebars](https://img.shields.io/badge/Handlebars-Templating-orange)
![JavaScript](https://img.shields.io/badge/JavaScript-Frontend-yellow)
![Status](https://img.shields.io/badge/Status-Complete-brightgreen)

CoffeeOverflow is a developer-themed coffee storefront built with Node.js, Express, and Handlebars.

The application simulates a modern online coffee shop experience including product browsing, cart management, checkout, user authentication, and membership subscriptions.

Designed with a dark developer-centric aesthetic, the project combines a full shopping flow with a playful programming theme.

![Homepage](Screenshots/01.%20HomepageHero.png)

---

# Features

- Dynamic coffee product catalogue
- Shopping cart system
- Checkout and payment simulation
- User authentication (login and signup)
- User profile management
- Membership subscription system
- Customer testimonials
- Contact form
- Developer themed UI design

---

# Tech Stack

### Backend
- Node.js
- Express.js

### Templating
- Express Handlebars

### Frontend
- HTML
- CSS
- JavaScript

### Data
- JSON product dataset

# Installation
Clone the repository
- git clone https://github.com/daniel-mcnamee-dev/CoffeeOverflow.git

Navigate into the project
- cd CoffeeOverflow

Install dependencies
- npm install

---

# Running the Application

Start the server
-npm start
or
- node app.js

Open in your browser
- http://localhost:3000
